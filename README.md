A real-time, full-stack ride booking platform that allows users to book rides, track captains, and manage trips. Inspired by Uber, this platform features live geolocation, captain login, and dynamic ride updates using WebSockets.

## 🧰 Tech Stack

### ⚛️ Frontend
- React.js
- Axios
- Gomaps (for maps)
- GSAP (animations)
- React Router DOM
- Tailwind CSS
- Socket.IO Client

### 🛠️ Backend
- Node.js
- Express.js
- MongoDB (Atlas)
- Mongoose
- Socket.IO
- bcrypt (password hashing)
- JSON Web Token (JWT)
- dotenv

### ☁️ Deployment
- Vercel (Frontend)
- Render (Backend)
- MongoDB Atlas (Database)
- GitHub (Version Control)
